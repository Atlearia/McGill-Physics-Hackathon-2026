# The Ten Commandments

Physics-based puzzle sandbox built for the McGill Physics Hackathon.

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Markup shell (canvas + script/style links) |
| `style.css` | Layout & dark theme |
| `config.js` | Constants & math/utility helpers |
| `physics.js` | Force application, impulses, wall helpers |
| `rendering.js` | Draw helpers, sidebar glyphs, main draw loop |
| `tools.js` | 6 tool definitions + maze wall segments |
| `game.js` | Game state, canvas setup, input, update, game loop |

## Run

Open `index.html` in a browser. No build step, no dependencies.
